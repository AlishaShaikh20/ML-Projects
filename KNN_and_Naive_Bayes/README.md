
# KNN and Naive Bayes Classification & Logistic Regression

## 📌 Project Overview
This project demonstrates the implementation and comparison of three popular machine learning classification algorithms:

- K-Nearest Neighbors (KNN)  
- Gaussian Naive Bayes
- Logistic Regression   

All models were trained and tested on the Iris Flower dataset to classify flower species based on flower measurements.

## Objective
To understand how different classification algorithms work and compare their performance on the same dataset.

##  Dataset Used
**Iris Flower Dataset**

### Features:
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

### Target:
- Iris Flower Species

## Workflow
- Imported dataset
- Performed data preprocessing
- Applied Label Encoding to target classes
- Split dataset into training and testing sets
- Trained Logistic Regression model
- Trained KNN model
- Trained Gaussian Naive Bayes model
- Evaluated models using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

---

##  Models Used

###  Logistic Regression
A supervised learning classification algorithm used for predicting categorical outputs.

###  K-Nearest Neighbors (KNN)
A distance-based algorithm that classifies data points based on nearest neighbors.

###  Gaussian Naive Bayes
A probabilistic classification algorithm based on Bayes Theorem.
