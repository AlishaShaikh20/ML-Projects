# E-commerce Purchase Prediction

# Objective
This machine learning project predicts whether a customer will generate revenue on an e-commerce platform.

## Algorithms Used
 Decision Tree Classifier

## Techniques Used

* Data Preprocessing
* OneHotEncoding
* StandardScaler
* Pipeline
* GridSearchCV
* Cross Validation
* Hyperparameter Tuning

## Evaluation Metrics

* F1 Score
* Classification Report
* Confusion Matrix

## Results

Best F1 Score: 0.634

Best Parameters:
 max_depth = 4
 min_samples_leaf = 50

* Python
* Pandas

* Jupyter Notebook
